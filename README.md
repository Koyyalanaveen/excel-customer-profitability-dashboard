# Customer Profitability & Sales Dashboard (Excel)

## Project Overview
This project presents an advanced Excel analytics solution to evaluate customer profitability, sales performance, and discount impact over a 3-year period.

## Tools & Techniques Used
- Microsoft Excel
- PivotTables & PivotCharts
- KPI Cards
- Slicers (Year, Region, Category)
- Conditional Formatting
- Dynamic Dashboards

## Key Insights
- Identified top and bottom profitable customers
- Analyzed sales and profit trends over time
- Evaluated discount effectiveness on profitability
- Highlighted high-risk and high-value customer segments

## File Included
- Customer_Profitability_3_Y.x.xlsb

## Outcome
This dashboard helps decision-makers quickly understand revenue drivers, profit trends, and discount strategies using a single interactive Excel interface.
